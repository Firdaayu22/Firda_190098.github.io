    Integrasi Numerik¶

    Integrasi Numerik merupakan suatu cara untuk menghitung aproksimasi luas daerah di bawah suatu fungsi.

    Integral terbagi menjadi 2, yakni Integral tak tentu dan Integral tentu.

    Integral Tak Tentu

    Integral tak tentu merupakan suatu fungsi yang membentuk suatu fungsi baru dan belum memiliki nilai pasti. Contoh (1.1) :

    \[ \int xdx= \int \dfrac{x^{2}}{2}+ C \]

    Integral Tentu

    Integral tentu merupakan suatu fungsi yang memiliki nilai pasti. Contoh (1.2):

    \[ \int^{1}_{0} xdx= \int^{1}_{0} \dfrac{x^{2}}{2}= \dfrac{1}{2} \]

    Metode Upper dan Lower Sum¶

    Metode Upper dan Lower Sum merupakan metode integrasi yang dilakukan dengan membagi interval menjadi 2 subinterval.

    \[ PartisiP=\{a=x_{0}\leq x_{1}\leq x_{2}\leq...\leq x_{n}=b\} \]

    Definisi :

    \[ m_{i}=min\{f(x): x_{i}\leq x\leq x_{i+1}\} \]
    \[ M_{i}=max\{f(x):x_{i}\leq x\leq x_{i+1}\} \]

    Metode Upper Sum dapat ditulis dengan Persamaan (2.1) :

    \[ U(f,P)=\sum^{n-1}_{i=0}M_{i}(x_{i+1}-x_{i}) \]

    Metode Lower Sum dapat ditulis dengan persamaan (2.2) :

    \[ L(f,P) = \sum^{n-1}_{i=0}m_{i}(x_{i+1} - x_{i}) \]

    Estimasi Integral :

    \[ Estimasi Integral = \dfrac {L+U}{2} \]

    Error :

    \[ Error \leq \dfrac {U-L}{2} \]

    Algoritma Metode Upper dan Lower Sum :

    1. Menentukan fungsi f(x) , interval (a,b) dan sub interval n.
    2. Hitung nilai antar interval.
    3. Hitung lower sum berdasarkan persamaan 2.2.
    4. Hitung upper sum berdasarkan persamaan 2.1.
    5. Jumlahkan hasil yang telah diperoleh untuk mendapatkan nilai estimasi integral.
    6. Hitung nilai error jika diperlukan.

    Contoh (2.1) :

    \[ \int^{b}_{a} x^2dx \]

    Import modul

    Import modul-modul yang diperlukan. Modul yang diperlukan pada contoh kali ini ada 2 modul, yakni modul numpy dan matplotlib.

    import numpy as np
import matplotlib.pyplot as plt

    Main program

    Setelah selesai import modul, selanjutnya membuat main program.

    def func(x):
    return (x**2)
def LU(a,b,n):
    p = [0,1/4,1/2,3/4,1]
    low = 0.0
    for i in range (n):
        low += func(p[i])

    up = 0.0
    for j in range ((len(p)-n), n+1):
        up += func(p[j])

    kali = p[1] - p[0]
    lower = kali*low
    upper = kali*up

    print('Lower sum         :', lower)
    print('Upper sum         :', upper)

    Tambahkan code di bawah ini dalam main program untuk estimasi integral dan error.

    estimasi = 0.5*(b-a)*(upper+lower)
error = 0.5*(b-a)*(upper-lower)

print('Estimasi Integral :', estimasi)
print('Error             :', error)

    Metode Newton-Cotes¶

    Metode Newton-Cotes merupakan metode integrasi yang dilakukan dengan membagi area di bawah kurva berdasarkan batas atas dan batas bawah.

    Terdapat beberapa metode dalam metode Newton-Cotes, yakni:

    1. Metode Trapezoid
    2. Metode Rekursif Trapezoid
    3. Metode Simpson ⅓

    Metode Trapezoid¶

    Metode Trapezoid merupakan metode integrasi yang dilakukan dengan melakukan pendekatan dengan area di bawah kurva menggunakan trapesium.

    Metode Trapezoid dapat ditulis dengan Persamaan (3.1) :

    \[ \int^{b}_{a} f(x)dx = \dfrac{1}{2} \Delta x(f(x_{0}) + 2 \sum^{n-1}_{i=1} f(x_{i}) + f(x_{n})) \]

    Dimana :

    \[ \Delta x = \dfrac{b-a}{n-1} \]

    Error dapat diestimasi menggunakan Persamaan (3.2) :

    \[ error = - \dfrac {b-a}{2}(f(x_{0})+f(x_{n-1})) \]

    Error pada setiap iterasi :

    \[ error = (f(x_{n-1})-f(x_{n})) \]

    Algoritma Metode Trapezoid :

    1. Menentukan fungsi f(x) , interval (a,b) dan sub interval n.

    2. Hitung nilai antar interval.

    3. Hitung jumlah nilai f(x) lalu dikali 2.

    4. Hitung hasil berdasarkan persamaan (3.1) dengan menjumlahkan nilai yang diperoleh.

    5. Hitung estimasi error dan error pada setiap iterasi jika diperlukan.

    Contoh (2.1):

    \[ \int^{b}_{a} x^2dx \]

    Import modul

    Import modul-modul yang diperlukan. Modul yang diperlukan pada contoh kali ini ada 2 modul, yakni modul numpy dan matplotlib.

import matplotlib.pyplot as plt #untuk visualisasi

    Setelah selesai import modul, selanjutnya membuat main program.

    def func(x):
    return (x**2)
def trap(a,b,n):
    dx = (b-a)/(n-1)
    x = np.linspace(a,b,n)

    sum = 0.0
    for i in range (1, n-1):
        sum += func(x[i])

    hasil = 0.5*dx*(func(x[0])+2.0*sum+func(x[-1]))

    print('Hasil :',hasil)

    Tambahkan code di bawah ini dalam main program untuk cek error

    print('Error :', hasil - (np.abs(-(0.5*(b-a)*(func(x[0])+func(x[-1]))))))

    Error pada setiap iterasi :

    for i in range(n-1):
        print('Error :', np.abs(func(x[i+1])-func(x[i])))

    Metode Recursive Trapezoid¶

    Metode Recursive Trapezoid dapat ditulis dengan Persamaan (4.1) :

    \[ R(n,0)= \dfrac {1}{2}R(n-1,0)+h[\sum^{2^(n-1)}_{k=1}f(a+(2k-1)h)] \]

    Dimana :

    \[ h = \dfrac {b-a}{2^n} \]

    Estimasi Error :

    \[ EstimasiError = |R(n,0)-R(n-1,0)| \]

    Algoritma Recursive Trapezoid :

    1. Menentukan fungsi f(x) , interval (a,b) dan sub interval n.

    2. Hitung nilai antar interval h.

    3. Hitung jumlah nilai f(a).

    4. Hitung hasil berdasarkan persamaan (4.1) dengan menjumlahkan nilai yang diperoleh dengan nilai sebelum-sebelumnya.

    5. Setelah itu hitung estimasi errornya.

    Contoh (4.1) :

    \[ \int^{\dfrac {\pi}{2}}_{0} x^2 dx \]

    def func(x):

def trapezoid(a,b,n):
    hasil=[]
    #estimasi awal
    h1 = (b-a)
    sum = h1/2*(func(a)+func(b))
    hasil.append(sum)

    #iterasi estimasi
    for i in range (1,2**(n-1)):
        h2 = h1/2**i
        if i == 1:
            x = i*h2
            sum = sum/2 + h2*func(x)
            hasil.append(sum)
        else:
            jml = 0
            #n genap
            if i % 2 == 0:
                for j in range (1,i+1):
                    batas = ((2*j) - 1)
                    x = batas*h2
                    jml += func(x)

            #n ganjil
            else:
                for j in range (1,i+2):
                    batas = ((2*j) - 1)
                    x = batas*h2
                    jml += func(x)
            sum = sum/2 + h2*jml
            hasil.append(sum)

    #print hasil akhir
    print("\nHasil Akhir =",sum)

    Metode Simpson ⅓¶

    Metode Simpson ⅓ merupakan metode yang perhitungannya menggunakan polynomial berderajat dua. Metode ini dinamakan metode Simpson ⅓ karena pada metode ini mengandung koefisien ⅓ di bagian depan persamaan tersebut.

    Metode Simpson ⅓ dapat ditulis dengan Persamaan (5.1) :

    \[ \int^{b}_{a} f(x)dx = \dfrac {h}{3} (f_{0} + 4 \sum^{n-1}_{i=1,3,5,...}f_{i}+2\sum^{n-2}_{i=2,4,6,...}f_{i}+f_{n}) \]

    Dimana :

    \[ h = \dfrac{b-a}{n} \]

    Algoritma Metode Simpson ⅓ :

    1. Menentukan fungsi f(x) , interval (a,b) dan sub interval n.

    2. Hitung nilai antar interval h.

    3. Hitung nilai f(a) dan f(b).

    4. Pisah antara yang ganjil dan yang genap.

    5. Jika f(x) ganjil maka akan dikali 4, 4 x f(x).

    6. Jika f(x) genap maka akan dikali 2, 2 x f(x).

    7. Hitung hasil berdasarkan persamaan (5.1) dengan menjumlahkan nilai yang diperoleh.

    Contoh (1.1) :

    \[ \int^{b}_{a} x^2dx \]

    Import modul

    Import modul-modul yang diperlukan. Modul yang diperlukan pada contoh kali ini hanya 1 modul, yakni modul numpy.

    import numpy as np

    Main program

    Setelah selesai import modul, selanjutnya membuat main program.

    def func(x):
    return (x**2)

def simpson(a,b,n):
    h = (b-a)/n
    n = n+1
    x = np.linspace(a,b,n)

    genap = 0.0
    for j in range (2, n-2, 2):
        genap += func(x[j])

    ganjil = 0.0
    for i in range (1, n-1, 2):
        ganjil += func(x[i])

    hasil = h/3*(func(x[0])+(4*ganjil)+(2*genap)+func(x[-1]))

    print('Hasil :', hasil)

    Tes Program

    a = 1.0
b = 10.0
n = 20
simpson(a,b,n)

    Apabila berhasil maka akan muncul hasil sebagai berikut:

    Hasil : 333.0
